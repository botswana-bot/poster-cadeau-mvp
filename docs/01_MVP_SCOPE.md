# MVP scope — ce qui est IN / OUT

## IN (MVP)
- Site FR
- France métropolitaine uniquement
- Produit : Poster A4, papier standard
- Prix produit : 19€
- Livraison : en plus (calculée plus tard via POD)
- Choix style (6 styles)
- Tagline (obligatoire, 60 caractères max)
- Ton tagline : gentil / sarcastique
- Prénom optionnel (pour crédits ou variations)
- 2 previews avant achat

## OUT (plus tard)
- A3 / 50x70
- Cadres inclus (upsell plus tard)
- Packs (6 variantes / pack cadeau)
- Génération automatique de taglines (IA)
- Comptes utilisateurs / historique
- International
- Galerie publique
- Paiement réel + POD (dans la démo)

## Anti‑abus / qualité (règles produit)
- 2 previews incluses, pas de regen illimitée.
- Après paiement : 1 génération finale HD incluse (+ 1 retry si bug technique).
- Preview basse résolution + watermark.

## “Photoréaliste pur” (interprétation pratique)
Même si l’objectif marketing est “photoréaliste pur”, en pratique on cherchera :
- rendu photo studio / ciné
- composition poster
- pas “identique pixel‑perfect” (souvent refusé)
- **ressemblance forte** mais sans formulation “exactly identical”.
