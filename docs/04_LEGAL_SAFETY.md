# Légal / sécurité / modération (notes)

## Consentement & droit à l’image
- L’utilisateur doit certifier :
  - consentement de la personne
  - personne majeure
- Conserver une preuve (plus tard) : horodatage + texte du consentement + IP (selon conformité) + identifiant commande.

## Mineurs
- Recommandation MVP : **interdit** (plus simple, plus sûr).

## Célébrités / personnes publiques
- À cadrer dans les CGU (souvent source de demandes limites / restrictions fournisseur).

## Contenus interdits
- Nudité explicite, violence, haine.
- Usurpation / deepfake à but de tromperie.

## Risques “photoréaliste + ressemblance forte”
- Refus provider images
- Résultats uncanny → régénérations coûteuses
- Attentes élevées → SAV

## Anti‑abus
- Previews watermarkées
- Pas de HD avant paiement
- Limites sur regénérations
